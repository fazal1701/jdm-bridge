## üé® Image Specifications

### General Guidelines
- **Format**: WebP (with JPG fallback)
- **Compression**: Optimized for web (70-80% quality)
- **Responsive**: Provide 1x, 2x, 3x versions for retina displays
- **Alt text**: All images need descriptive alt text
- **Lazy loading**: Implement for below-fold images

### Naming Convention
```
{page}-{section}-{description}-{size}.webp
Example: marketplace-hero-skyline-gtr-1920x1080.webp
```
